# Librariarr

Media library management for Plex, Jellyfin, and Emby.

**Documentation:** [librariarr.dev](https://librariarr.dev)

**Source:** [github.com/ahembree/librariarr](https://github.com/ahembree/librariarr)

## Features

- **Multi-server support** — Connect multiple Plex, Jellyfin, and Emby servers
- **Plex OAuth** — Secure authentication via your Plex account
- **Library browsing** — Filterable, searchable views for movies, series, and music
- **Lifecycle rules engine** — Automated media management with recursive AND/OR rule groups, configurable actions (delete, unmonitor, tag), and scheduling
- **Stream Manager** — Monitor active sessions, maintenance mode, transcode management, and blackout schedules
- **Preroll Manager** — Manage Plex preroll presets and schedules
- **Arr integration** — Sonarr, Radarr, and Lidarr support for lifecycle actions
- **Discord notifications** — Webhook alerts for lifecycle events and maintenance changes
- **Backup & restore** — Full or config-only backups with optional AES-256-GCM encryption
